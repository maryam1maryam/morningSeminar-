# Основы работы Git #

## Репозиторий - 
+ *место, где хранятся и поддерживаются какие-либо данные*

## Система контроля версий - 
+ ***это система, регистрирующая изменения в одном или нескольких файлах***

***Виды систем контроль версий:***

2. Централизованные
3. Распределенные
